  • ⚠️ og:locale غير موجود (defaults to en_US)
  • ⚠️ og:site_name غير موجود

🔗 Canonical & Hreflang

❌ لا يوجد canonical tag في أي صفحة
❌ لا يوجد hreflang tags
  • ⚠️ يجب إضافة canonical لكل صفحة
  • ⚠️ يجب إضافة hreflang="ar" للعربية
  • ⚠️ يجب إضافة alternate hreflang للغة الإنجليزية

🏷️ Schema Markup

❌ لا يوجد أي Schema Markup!

المطلوب إضافته:

  • Organization Schema: لربط الموقع بالمنظمة
  • LocalBusiness Schema: للموقع الفعلي (بحرة)
  • WebSite Schema: مع searchAction
  • NGO Schema: (إن وجد)
  • ImageObject: للصور
  • BreadcrumbList: للتنقل
💡 Schema.org NGO type غير موجود مباشرة، يمكن استخدام Organization أو CharitableOrganization

🖼️ تحسين الصور

  • ⚠️ لم نتمكن من فحص alt texts (JavaScript rendering)
  • ⚠️ الصور مستضافة على Azure Blob - لا نعرف الأبعاد
  • ⚠️ لا يوجد lazy loading ظاهر
  • 💡 تأكد من أن جميع الصور لها alt text
  • 💡 يفضل ضغط الصور قبل الرفع
  • 💡 استخدام WebP format إن أمكن

❌ المشاكل On-Page

1. لا Canonical Tags
يجب إضافتها لجميع الصفحات
2. لا Schema Markup
فرصة ضائعة للسيRich Snippets
3. Meta Description مفقود في صفحات المشاريع
صفحات فرادية تفتقر للوصف
4. OG:image بدون أبعاد
يجب إضافة og:image:width و og:image:height
5. لا og:locale
يجب تحديد ar-SA
6. H1/H2 hierarchy غير واضح
بسبب SPA، يصعب تحليل الهيدر

✅ توصيات On-Page

  • عالية: إضافة canonical tag لكل صفحة
  • عالية: إضافة Schema Markup (Organization, LocalBusiness, WebSite)
  • عالية: إضافة meta description لصفحات المشاريع
  • متوسطة: إضافة og:image:width و og:image:height
  • متوسطة: إضافة og:locale="ar_SA"
  • متوسطة: إضافة og:site_name
  • متوسطة: تحسين image alt texts
  • متوسطة: تحسين header hierarchy
  • ثانوية: إضافة hreflang tags
